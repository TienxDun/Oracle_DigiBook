Triggers TR 1 – TR 3

Kiểm thử 3 trigger: validation (chặn dữ liệu sai), tính lại total_amount tránh mutating table, và ghi audit log theo từng action.

TR 1

Validation Trigger

Chạy expected-fail case — trigger BEFORE INSERT/UPDATE phải chặn các trường hợp dữ liệu không hợp lệ. Kết quả PASS khi trigger chặn đúng.

Chưa chạy test trigger validation.

TR 2

Recalculate Trigger

Tạo đơn test tạm thời, thêm / sửa / xóa order_details và đối chiếu total_amount sau từng bước.

Chưa chạy test trigger recalculate.

TR 3

Audit Log Trigger

Tạo đơn hàng, thực hiện UPDATE rồi DELETE — kiểm tra audit log ghi đúng INSERT / UPDATE / DELETE.

Chưa chạy test trigger audit.
